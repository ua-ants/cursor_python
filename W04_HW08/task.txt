Установите Flask
pip install Flask
2. Создайте маленькое приложение на Flask

создайте файл app.py

from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'
Откройте терминал и перейдите в директорию с файлом  app.py. Выполните:

$ export FLASK_APP=hello.py
$ flask run
 * Running on http://127.0.0.1:5000/

Перейдите в браузере на страничку localhost:5000, Вы должны увидеть надпись 'Hello, World'. Если надпись не появилась, проверьте, что приложение запустилось на 5000 порту.



Не забывайте перезапускать сервер командой flask run, когда меняете файл app.py
